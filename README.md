# Asset Complaint Hub

A web application designed to manage the lifecycle of IT asset-related complaints within an organization. It allows users to report issues and administrators to efficiently track, manage, and resolve them.

## Features
- Create and manage complaints for IT assets
- Track complaint status (NEW, IN_REVIEW, IN_PROGRESS, RESOLVED, CLOSED, REJECTED)
- Comment system for each complaint
- User, department, and asset management
- PDF report generation
- Statistics dashboard

## Technologies Used

### Frontend
- Angular 21
- Angular Material + CDK
- TailwindCSS
- Chart.js
- RxJS

### Backend
- Java 21
- Jakarta EE (Web API)
- Maven
- Lombok
- JWT (authentication)
- BCrypt (password hashing)
- PostgreSQL Database
- Jackson (JSON processing)
- Swagger / OpenAPI
- iText (PDF generation)
- WildFly (application server
