# Asset Complaint Hub

Aplicatie web pentru gestionarea ciclului de viata al plangerilor asociate asset-urilor IT dintr-o organizatie. Permite utilizatorilor sa raporteze probleme, iar administratorilor sa le gestioneze si sa le rezolve eficient.

## Functionalitati
- Creare si gestionare plangeri pentru asset-uri
- Vizualizare status (NEW, IN_REVIEW, IN_PROGRESS, RESOLVED, CLOSED, REJECTED)
- Sistem de comentarii pentru fiecare plangere
- Administrare utilizatori, departamente si asset-uri
- Generare rapoarte (PDF)
- Dashboard cu statistici

## Tehnologii folosite

### Frontend
- Angular 21
- Angular Material + CDK
- TailwindCSS
- Chart.js
- RxJS

### Backend
- Java 21
- Jakarta EE (Web API)
- Maven
- Lombok
- JWT (autentificare)
- BCrypt (hash parole)
- PostgreSQL Database
- Jackson (JSON)
- Swagger / OpenAPI
- iText (Rapoart PDF)
- WildFly (application server)
